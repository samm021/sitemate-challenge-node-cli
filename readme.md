# SITEMATE CHALLENGE CLI

![CLI view](https://imgur.com/a/ItP3qMB)

## requirements

- node v20.3.0
- npm v9.6.7

## dependencies

- readline-sync
- dotenv

## to start

1. sitemate-challenge-server runnning in background (https://github.com/samm021/sitemate-challenge-server);
2. open terminal / command line
3. `npm install`
4. `npm run start`
